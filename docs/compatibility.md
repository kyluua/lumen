# Compatibility Matrix

| Area | Supported |
| --- | --- |
| Minecraft | 1.21.x first |
| Lumen runtime lines | 26.1.x, 26.2.x |
| Loaders | Vanilla, Fabric, Forge, NeoForge |
| Java | JDK 21 baseline, JDK 25 supported |
| OS | Windows 10/11, macOS, Linux desktop JavaFX targets |

Lumen treats 26.1.x and 26.2.x as Lumen release/runtime compatibility lines unless the target ecosystem defines them differently.
