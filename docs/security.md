# Security and Safety Policy

Lumen is a legitimate launcher and client-management app.

It will not include cheats, hacks, anti-cheat bypasses, unfair automation, exploit modules, X-ray, kill aura, scaffold, packet abuse, server bypass configs, cracked account flows, account stealing, token scraping, hidden network behavior, piracy, or evasion tooling.

Network calls are explicit and logged. Downloads prefer checksums/signatures where providers publish them. Extensions are limited to themes, launch hooks, and UI widgets.
